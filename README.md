 # Mobile User Authorisation

 # Вашему вниманию представлено приложение "First", которое реализует регистрацию пользователя в личном кабинете.

 ## Принцип работы приложения

После запуска приложения пользователь вводит логин и номер телефона, после чего нажимает кнопку "Зарегистрироваться".
Если указанные поля ввода заполнены, то приложение перейдет на стричку "Информация о пользователе", в противном случае будет выведено предупреждающее сообщение "Заполните поля ввода". 
В случае успешного перехода на страничку "Информация о пользователе" необходимо заполнить поля Ф.И.О., интересы, согласия и т.д. 
После заполнения данных, пользователю необходимо нажать кнопку приметить, после чего информация о пользователе будет выведена в режиме "Отладки" программы.
